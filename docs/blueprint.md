# **App Name**: WebbyTalk

## Core Features:

- Message Reception: Receive incoming messages from the webhook URL.
- Message Display: Display messages in a chronological order.
- Markdown formatting: Format messages using markdown.
- Sentiment Analysis: Use AI to categorize the sentiment of incoming messages
- Display Message Sentiment: Display message sentiment.
- Conversation summarization: AI tool summarizes recent conversation. The summarization frequency should intelligently adjust over time based on conversation patterns.  

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a sense of sophistication and focus.
- Background color: Very light purple (#F0F8FF) for a calming and unobtrusive backdrop.
- Accent color: Violet (#8A2BE2) to draw attention to the latest messages, and the summarization.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text due to its modern, neutral, and readable design.
- Use minimalist icons for displaying the message source, date, time, sentiment.
- Simple and clean layout with a focus on readability, placing key actions where users expect.
- Subtle animations for new messages appearing.
